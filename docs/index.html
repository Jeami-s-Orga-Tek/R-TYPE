Menu & gameplay

What’s inside

Graphical Client (SFML)

Clean UI flow Launch → Menu → Lobby → Parameters → Play. Smooth starfield, particles, embedded fonts & sounds. Remappable controls and 30/60 FPS caps.

Authoritative UDP Server

Asynchronous networking with Boost.Asio: binary handshake, ping/heartbeat, welcome & lobby messages. Server remains stable even if a client hiccups.

Custom Game Engine

Entity‑Component‑System core with a Mediator to decouple subsystems (render, physics, input, net). Reusable managers for Entities, Components, and Systems.

Co‑op Ready

Designed for up to 4 players (Solo/Duo/Trio/Squad). Distinct player identity and missile systems on both sides.

Settings & Accessibility

Resolution & display mode, graphics quality, color‑blind profiles, audio volume, controls page to tweak inputs.

DX‑free Build

Self‑contained builds via CMake + CPM.cmake. Dependencies are fetched on first configure and binaries land at the repo root.

Gameplay snapshot

Core loop

  • Fly forward, dodge bullets and hazards.
  • Fire missiles (player & enemy); collision & damage.
  • Pickup power‑ups (WIP), clear waves, beat the boss (WIP).

Default controls

  • Move: Arrow keys
  • Fire / Special: configurable in Parameters → Controls
  • FPS cap: toggle 30/60 in Parameters

Build & Run

Requirements

  • Linux (tested) or Windows (MSVC, WIP)
  • CMake ≥ 3.10, a C++17 compiler
  • Internet access for first-time dependency fetch (CPM.cmake)

Linux quick start

cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build -j
./r-type_server &
./r-type_client

Binaries are produced at the repository root.

Windows (MSVC, WIP)

cmake -S . -B build -G "Visual Studio 17 2022" -A x64
cmake --build build --config Release
r-type_server.exe  &
r-type_client.exe

Networking protocol

sequenceDiagram participant C as "Client" participant S as "Server" C->>S: "HELLO (connect)" S-->>C: "WELCOME (playerId)" loop "Heartbeat" C->>S: "PING(seq)" S-->>C: "PONG(seq)" S-->>C: "SNAPSHOT(state)" end

Server is authoritative; heartbeats keep connections healthy and drive world snapshots.
